# Gesture Control App

A modular Python application for controlling your computer using hand gestures.

## Structure

- `app.py`: Main entry point.
- `core/`: Core logic (tracking, gesture detection, actions).
- `config/`: Configuration settings.
- `utils/`: Helper utilities.

## Setup

1. Install dependencies:
   ```
   pip install -r requirements.txt
   ```
2. Run the app:
   ```
   python app.py
   ```
